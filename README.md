# RADAR FAIR Data Point

The data point is based on one that was created as result of the TranSMART FAIR Hackathon in The Hyve.

## Run the datapoint

First you need to change RADAR connection details in the `radar_fair_metadata.py` file.
The code is written wity Python 3. Two python packages were used: `rdflib` and `falcon`.
On how to run the falcon web application see the [falcon website](https://falconframework.org/).
